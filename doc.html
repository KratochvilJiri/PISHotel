Vyžadované knihovny

Nebyli použity žádné speciální knihovny, pouze frameworky/technologie. Ty jsou uvedeny v souborech package.json (backend technologie), respektive bower.json (frontend technologie).

Instalace a spuštění

  • Prerekvizity (nutné předem nainstalovat): node.js, bower, npm.
  • Terminál - kořenový adresář aplikace - příkaz npm install - instalace backend technologií.
  • Terminál - kořenový adresář aplikace - příkaz bower install - instalace frontend technologií.
  • Terminál - kořenový adresář aplikace - příkaz node server.js - spuštění serveru, aplikace nyní přístupná z http://localhost:8080/
